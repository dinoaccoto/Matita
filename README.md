# Matita IDE

**Matita** è un ambiente di sviluppo integrato (IDE) educativo, leggero e bilingue, contenuto in un singolo file HTML. È progettato per insegnare i concetti di programmazione e il pensiero algoritmico utilizzando comandi sia in **italiano** che in **inglese**.

## Caratteristiche

- **Single-File:** basta aprire il file nel browser. Si sconsiglia l'uso da smartphone per avere sufficiente area di lavoro.
- **Bilingue:** L'interprete accetta comandi misti o specifici in Italiano (`se`, `fintanto`, `disegna`) e Inglese (`if`, `while`, `draw`).
- **Grafica Integrata:** Include una finestra Canvas mobile e ridimensionabile per disegnare primitive geometriche, immagini e testo.
- **Editor:**
  - Evidenziazione della sintassi (Syntax Highlighting).
  - Numerazione delle righe.
  - Autocompletamento intelligente (Suggerimenti).
- **Strumenti:** Console di output, salvataggio/caricamento file locale (.txt, .matita), ed esportazione screenshot PNG.

## Come usarlo

1. Scarica il file `index.html` (o `matita.html`) da questo repository.
2. Apri il file con un browser moderno (Chrome, Edge, Firefox, Safari).
3. Inizia a scrivere codice nell'editor a sinistra.
4. Premi **Esegui** (o `Ctrl + Enter`) per avviare il programma.

Si consiglia di scaricare anche i seguenti file:

esempiIT.html (esempi di programmi in Matita)
help_it.html (guida al linguaggio)
infoIT.html (caratteristiche di Matita)
storiaIT.html (motivazione)
matita.pdf (versione PDF della documentazione)
